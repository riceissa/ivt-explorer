Theorem (Intermediate value theorem). Let a, b be real numbers such that a < b. Let f: [a,b] → R be a continuous function such that f(a) < 0 and f(b) > 0. Then there exists c ∈ [a,b] such that f(c) = 0.

How do we prove this theorem? There are several approaches, but here we will directly use the least upper bound property of the real numbers, which says that any nonempty set of real numbers which is bounded above has a least upper bound.

Let's try to directly find a root of the function f. The displayed function has multiple such roots, so we'll have to pick one of them to find. Which one do you want to find?

  • Find the smallest (leftmost) root
  • Find the second smallest root
  • Find the penultimate (second from the right) root
  • Find the largest (rightmost) root

Good choice! What expression finds the smallest root?

  • sup{x ∈ [a,b] : f(x) < 0}
  • define Vx to be Vx = f([a,x]) = {f(x) : x ∈ [a,x]}. Then the root in question is sup{x ∈ [a,b] : sup Vx ≤ 0}
  • define Vx to be Vx = f([a,x]) = {f(x) : x ∈ [a,x]}. Then the root in question is sup{x ∈ [a,b] : sup Vx < 0}

That will work. So let c := sup{x ∈ [a,b] : sup Vx ≤ 0}. Now we have to show that this is actually a real number, i.e. that the least upper bound actually exists. To do this, we have to show that the set {x ∈ [a,b] : sup Vx ≤ 0} is nonempty and bounded above.

We stipulate that a is less than b, because the cases a=b and a > b are less interesting. If a=b, then the interval [a,b] is just the singleton set {a}, so the function consists of a single point (a, f(a)), which is not interesting. If a > b, then the interval [a,b] is empty, so the function is the empty function, so the theorem is vacuously true.
A root of a function f is a number x such that f(x) = 0.

The least upper bound property for the real numbers only works when the set S in question is nonempty and bounded above.

If the set is empty, i.e. S = Ø, then any number is an upper bound. For example, vacuously for any x ∈ S, we have x ≤ 1 so the number 1 is an upper bound—this works because there isn't a number in S, so anything we say about such a number is true. This works for any number, not just 1. This means that while there are upper bounds, there is no least upper bound, because for any upper bound, we can just subtract 1 to get an even smaller one.

On the other hand, if S is not bounded above, then there is no upper bound at all, so there can't be a least upper bound either.
